# House Price Prediction using Linear Regression

This project is a small machine learning project developed during my self-learning in AI.

The goal is to predict house prices using structured data.

## Dataset
Housing dataset with features such as:
- area
- number of rooms
- city
- location attributes

## Pipeline

1. Data loading using pandas
2. Data cleaning (handling missing values)
3. Encoding categorical variables
4. Train/test split
5. Linear regression training
6. Evaluation using R² score

## Run

Open the notebook:

jupyter notebook regression.ipynb
